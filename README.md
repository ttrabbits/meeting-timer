# meeting-timer

- シンプルなカウントダウンタイマー
- 会議のアジェンダ (担当者・持ち時間) を設定できる
- https://ttrabbits.github.io/meeting-timer

## Features

- カウントダウンタイマー
- 終了予定時刻の表示
- 予定リストの作成
- 終了時・時間超過時のサウンド再生
- クエリパラメータによる状態の保存

## Setup

Requirements: Node >= 20

```sh
pnpm install
```

## Development

```sh
# 開発サーバー
pnpm dev

# テスト
pnpm test

# Lint / Format
pnpm lint
pnpm format

# ビルド
pnpm build
```
